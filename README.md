## A modern, AI-powered civic guidance application designed to help Indian voters navigate the complex electoral process. 

## Google Services

This application leverages three core Google Services:

1. **Gemini API**: Powers all semantic search intent classification and natural language explanations of complex civic procedures. The API uses structured JSON output to guarantee UI compatibility and employs strict safety filters (Harassment, Dangerous Content) to prevent misuse.
2. **Firestore**: The backend database used to securely store and structure the election guide's civic data, ensuring schemas are enforced.
3. **Firebase Analytics**: Tracks user interactions—such as task selection, external link clicks (source chips), flow completions, and AI fallbacks—to help continuously improve the user experience and findability of civic resources.
